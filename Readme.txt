///////////////////////////////////////////////////////////////
		Client Server App using Sockets
///////////////////////////////////////////////////////////////

This is a simple client server application through which the user can share files. The user can select to upload the file on either the CPP server or JAVA server. There are specific clients as well as servers written for both these platform therefore the user use either of the clients to send files to either of the servers.

The main method uses threads to start client as well as the server.
